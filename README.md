# 🏦 Python Bank Management System

A simple desktop-based banking system built using **Python** and **Tkinter**.  
This project provides basic banking operations through a graphical user interface (GUI).

---

## 📖 Overview

The application simulates a small banking system where users can:

- Create accounts
- Deposit money
- Withdraw money
- Transfer money
- View balances
- Reset PINs
- Manage multiple accounts

The project is implemented using:

- **Python**
- **Tkinter GUI**
- Object-Oriented Programming (OOP)

---

## ✨ Features

### 👤 Account Management
- Create new bank accounts
- Secure account access using a 4-digit PIN
- Delete existing accounts

### 💰 Transactions
- Deposit money
- Withdraw money
- Transfer money between accounts

### 🔐 Security Features
- PIN verification
- PIN reset using backup number

### 📊 Account Information
- View account balance
- View all created accounts

### 🖥️ Graphical User Interface
- Easy-to-use Tkinter interface
- Dialog-based interaction
- Desktop application experience

---

## 🛠️ Technologies Used

- **Python 3**
- **Tkinter**
- Object-Oriented Programming (OOP)
